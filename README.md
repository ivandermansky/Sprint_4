Серия автоматизированных тестов.
Проект для автоматизации тестирования веб-приложения с использованием Selenium.
Тесты выполняются с помощью WebDriver и проверяют функциональность UI-элементов сайта https://qa-scooter.praktikum-services.ru/.
Используемые технологии и зависимости:
Selenium Java (4.24.0) — основной фреймворк для автоматизации браузера.
WebDriver Manager (5.3.1) — утилита для автоматического управления драйверами браузеров (ChromeDriver, GeckoDriver и др.).
JUnit (4.13.2) — фреймворк для написания и запуска юнит-тестов (указан в двух зависимостях с разными scope).

Требования к окружению.
Для работы проекта необходимы:
Java JDK версии 11.
Maven (для сборки и управления зависимостями).
Браузер с установленным драйвером (Chrome).
IntelliJ IDEA для разработки и отладки.
Структура проекта:
src/main/java/ — основной код (классы главной страницы и страницы заказа).
src/test/java/ — тестовые классы и сценарии.
pom.xml — конфигурация Maven (зависимости, плагины).

Клонируйте репозиторий на локальный компьютер:
bash git clone <ссылка_на_репозиторий>
Откройте проект в IntelliJ IDEA.
Убедитесь, что в IDEA настроен JDK версии 11.
Выполните сборку проекта с помощью Maven: mvn clean install
Для запуска тестов используйте следующие команды:
Запуск всех тестов: mvn test
